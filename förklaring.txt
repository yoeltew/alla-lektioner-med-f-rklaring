# Lektion 2 - Listor och Loopar i Python
===================================

VAD HANDLAR LEKTIONEN OM?
------------------------
I denna lektion lär du dig om listor (arrays) och hur man kan upprepa kod med hjälp
av loopar. Detta är grundläggande koncept som används mycket i programmering.

HUVUDKONCEPT:
------------
1. Listor
   - En samling av värden
   - Exempel: hundar = ["bob", "woff", "sten"]
   - Kan innehålla olika typer av data

2. For-loopar
   - Upprepar kod för varje element i en lista
   - Två huvudsätt att använda for-loopar:
     a) För varje element: for hund in hundar
     b) Med range: for i in range(3)

3. Range-funktionen
   - Skapar en sekvens av nummer
   - range(3) ger [0,1,2]
   - Används ofta i loopar

4. Lista-metoder
   - append(): Lägger till element i slutet
   - len(): Får längden på listan
   - Åtkomst med index: lista[0]

PRAKTISKA EXEMPEL:
----------------
```python
# Lista med for-loop
hundar = ["bob", "woff", "sten"]
for hund in hundar:
    print(hund + " är en hund")

# Användning av range
for i in range(3):
    print("Hello")
    print(i)

# Lägga till i lista
nummer_lista = [7, 3, 25]
nummer_lista.append(543)
print(nummer_lista)
```
